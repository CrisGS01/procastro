__init__(data, coordsxy=None, labels=None, stamprad=20, maxskip=6, keydate='DATE-OBS', keytype='IMAGETYP', mastermode='median')[source]¶

Timeseries object constructor.

Parameters:
  • data (str (path to a folder), astrodir, ndarray list (list of images)) – image data for the timeseries
  • coordsxy (list of 2-tuples or 2-list) – list of x,y coordinates of stars in the first image of the timeseries (if the value is None an interface for coordinates selection will be displayed)
  • labels (list of strings) – labels for the stars
  • stamprad (int (default value: 20)) – radius of the (square) stamp that contains a star
  • maxskip (int (default value: 6)) – maximum skip considered (without warning) between the positions of a star in two sucesive images
  • keydate (str) – header key of observation date in the images (allows image sorting in terms of time)
  • keytype (str) – header key of image type (‘OBJECT’,’BIAS’,’DOME FLAT’,etc)
  • mastermode (str) – mode for master (BIAS, DARK, FLAT, etc) image obtention. Options: ‘mean’ and ‘median’ (default).
Return type:
